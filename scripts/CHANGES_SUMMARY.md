### 4. كيفية التشغيل:

#### للاختبار المحلي:
```bash
cd scripts
npm install
npm run test-local-auth
```

#### لإنشاء المستخدمين في Supabase:
```bash
# 1. إنشاء ملف .env.local أولاً
# 2. تثبيت المكتبات
npm install
# 3. اختبار الاتصال
npm run test-connection
# 4. إنشاء المستخدمين
npm run create-users
```
